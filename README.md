# site-traian-vuia

Website concept for **Salonul Internațional de Invenții și Inovație Traian Vuia 2026**.

## Structure
- `index.html`
- `css/style.css`, `css/animations.css`, `css/responsive.css`
- `js/main.js`, `js/three-scene.js`, `js/animations.js`, `js/countdown.js`, `js/translations.js`, `js/lang-switcher.js`
- `assets/logo-placeholder.svg`

## Notes
- Romanian is default language; 11-language switcher is included.
- Three.js and GSAP are loaded from CDN.
- All requested sections are present as placeholders and can be expanded with final content.
